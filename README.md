# Household Budget API

A Python-based backend API for calculating household budgets.

This project is a backend version of a household budget calculator, exposing the budget logic through a simple API instead of terminal input.

## Purpose
The purpose of this project is to practice backend development and API design using Python.

## Planned features
- Calculate household income
- Handle fixed and variable expenses
- Calculate loan costs
- Return budget summaries via API endpoints

## Tech stack
- Python
- Flask (planned)

## Project status
This project is under development and will be expanded step by step.
